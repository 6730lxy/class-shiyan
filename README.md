# ex04
完成基于触摸传感器的“自锁”开关
关联基础：实验2 (基础IO)、实验4 (触摸引脚)
提示：基础的触摸实验通常是“摸着亮，松开灭”，要求引入布尔型状态变量（如 `bool ledState`）和边缘检测逻辑（判断上一次未触摸，当前被触摸的瞬间）。
只有在检测到按下瞬间，才翻转LED的状态。
为了防止手抖导致的误触发，还需要加入软件防抖逻辑。
预期实验结果：摸一下触摸引脚，LED亮起并保持长亮；松开手再摸一下，LED熄灭

# ex05
完成多档位触摸调速呼吸灯
关联基础：实验3 (PWM呼吸灯)、实验4 (触摸引脚)
提示：结合呼吸灯的PWM代码和触摸自锁开关逻辑。定义一个变量表示“速度档位”（如 1、2、3档），每次触发触摸引脚，档位循环切换（1->2->3->1...）。
在PWM循环中，根据当前的档位变量，改变 `delay()` 的时长或占空比每次递增的步长（Step）。
预期实验结果：LED持续呈呼吸灯效果。
用户每触摸一次引脚，LED“呼吸”的节奏就会发生改变（如从缓慢呼吸变为急促呼吸，共有三个明显的速度级别）。

# ex06
完成警车双闪灯效（双通道PWM）
关联基础：实验3 (PWM呼吸灯)
提示：要求在面包板上连接两个LED灯至不同的引脚，初始化两个独立的PWM通道。
编写逻辑让两个灯的亮度呈现“反相”关系：当灯A的占空比从0增加到255时，灯B的占空比恰好从255减小到0。
预期实验结果： 两个LED灯呈现平滑交替的渐变闪烁效果，一个变亮的同时另一个逐渐变暗，过渡非常柔和。
